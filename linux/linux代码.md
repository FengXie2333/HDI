#  linux代码

`wc`：

-c 统计字节数。  

\- l 统计行数。  

\- w 统计字数  



`wget`从特定的URL上下载



`ls`

ls -lR 逐层显示



cat > cat1 

往 cat1 中逐行输入文本，用 Ctrl-D 结束输入 

cat >> cat1 

往 cat1 中逐行追加文本，用 Ctrl-D 结束输入 



less 

逐屏显示文件内容，回车进一行，空格进一页， q 终止显示 



rm –r temp/* 

删除子目录 temp 下所有子目录和文件，保留该目录 

rm –r temp 

删除子目录 temp 和该目录下所有子目录和文件
